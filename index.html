MÉTODO DE REQUISIÇÃO PADRÃO#

MENSAGEM DE RETORNO#

Todas as respostas devem ser no formato JSON com as seguintes propriedades

Propiedade Descrição
code código de retorno do webservice

DICIONÁRIO DE DADOS DA PROPRIEDADE CODE#

Valor Descrição
1 Método executado com sucesso e retornou o resultado
2 Falha de conexão com o banco de dados
3 Método não encontrado
